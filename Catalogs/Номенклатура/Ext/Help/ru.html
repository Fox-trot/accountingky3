Справочник Номенклатура

Классификатор номенклатур

Справочник Номенклатура представляет с собой информацию о номенклатурах. Для номенклатур используется Код МТО (Системный код не доступен для редактирования и не используется). Если создается новая номенклатура в филиале, то она создается с буквенным префиксом (первая буква филиала).

Классификатор представляет собой внешний файл mxl, содержащий номенклатуры, которая заполняется, изменяется в МТО. Классификатор при изменении обновляется на сервере.
Создание новой номенклатуры в классификаторе

Осуществляется только в МТО. При записи номенклатуры в справочнике номенклатуры, классификатор обновляется по команде «Обновить классификатор»

Подбор номенклатуры из классификатора

Фактически это перенос номенклатуры из классификатора в справочник номенклатуры.

Подобрать из классификатора можем как номенклатуру, подгруппу или группу. При этом если подобрана новая номенклатура, то в справочнике номенклатур программа формирует её подгруппу и группу (в иерархии). Если в Классификаторе случайно подобрана номенклатура, которая уже есть в справочнике номенклатур программа выводит сообщение о том, что номенклатура уже подобрана.
Обновление классификатора МТО при добавлении или редактировании классификатора обновляет его на сервере (по кнопке Обновить…)
