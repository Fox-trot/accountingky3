Номенклатура

Для хранения информации по всем товарам (услугам) предназначен справочник Номенклатура.

Так как в справочнике Номенклатура хранится информация по всем видам товарно-материальных запасов, будь то материалы, товары для продажи, готовая продукция, полуфабрикаты и прочие виды запасов целесообразно группировать их по видам, например: Товары, Материалы, Услуги и т.д. При первом запуске информационной базы в справочнике Номенклатура создаются группы по умолчанию: Товары, Услуги, Готовая продукция, Материалы, МБП.

В плане счетов учет ТМЗ организован на счете 1600 (Запасы), на этом счете открыты следующие синтетические счета:

  • 1610- Товары.
  • 1620- Запасы сырья и материалов.
  • 1640- Готовая продукция.
  • 1695- Прочие запасы.

В карточке элемента справочника Номенклатура отражается следующая информация:

Наименование- краткое наименование товара, для быстрого поиска нужной номенклатурной единицы желательно в это поле вносить краткую информацию без знаков "кавычки";
Наименование для печати- полное наименование товара, которое будет отображаться во всех печатных формах документов;
Код ТНВЭД- код товарной номенклатуры ВЭД;
Группа- в этом поле указывается группа, к которой будет принадлежать данная номенклатурная позиция;
Услуга- в справочнике Номенклатура, хранится информация не только по ТМЗ, в этом же справочнике указывается информация по приобретенным услугам (услуги сторонних организаций), по оказанным услугам, поэтому если номенклатурная позиция относится к типу Услуга необходимо установить флажок.

Как найти: Запасы → Справочники → Номенклатура

Видео инструкция: Сервис классификатор ГКЭД

Видео инструкция: Сервис классификатор кодов ТНВЭД
