Документ "Регламентированный отчет"

 

Документ, Регламентированный отчет в режиме пользователя представлен следующими видами финансовых отчетов:

·        Бухгалтерский баланс;

·        Отчет о прибылях и убытках;

·        Отчет о движении денежных средств;

·        Отчет об изменениях в собственном капитале.

1.  Документ предназначен для формирования и хранения соответствующего регламентного отчета

2.  Отчет формируется за заданный период, кратный месяцу. Если предыдущий период не задается, то он определяется как соответствующий период предыдущего года. Предыдущий период может быть задан вручную

3. Отчет формируется для заданной организации. Если организация не задана – то по предприятию в целом.

4. Настройка отчета задается в соответствующем  окне выбором из справочника «Настройки ОПУ». При выборе настройки автоматически заполняются настройки строк закладок ОТЧЕТ и НАСТРОЙКИ.

5. В каждом месяце может быть сформировано неограниченное число документов ОПУ с различными настройками.

6. Формирование отчета производится  по кнопке РАССЧИТАТЬ – заполняются закладки ОТЧЕТ, НАСТРОЙКИ и РАСШИФРОВКИ.

7. На закладке ОТЧЕТ показаны все строки отчета, включая заголовочные (не рассчитываемые) строки.  Для каждой строки указывается:

·        код строки;

·        строка-группа, в которую входит данная строка;

·        знак, с которым строка входит в группу (плюс или минус);

·        признак заголовка, если строка заголовочная;

·        номер прохода – для указания последовательности расчета групповых строк.

Настройки строки видимы при установленном флажке ПОКАЗАТЬ НАСТРОЙКИ. После расчета на закладке показываются суммы строк для отчетного и предшествующего периода.

8. На закладке НАСТРОЙКИ показывается расшифровка для текущей строки закладки ОТЧЕТ – формулы по счетам, из которых формируется строка – указываются:

·        код строки

·        счет, сумма по которому входит в строку

·        формула суммы для счета (обороты в дебет, обороты с кредита, сальдо дебетовое конечное, сальдо кредитовое конечное)

·        знак, с котором  сумма счета входит в строку (плюс не показывается)

По флажку ВСЕ НАСТРОЙКИ можно просмотреть полный список настроек для всех строк отчета

9. На закладке РАСШИФРОВКИ показываются суммы по счетам для всех строк. Кнопками отбора можно выделить строки для анализа и поверки.

10. Настройки отчета можно изменять непосредственно в документе, при этом можно использовать кнопку НОМЕРА ПРОХОДОВ для автоматического формирования номеров прохода. ВНИМАНИЕ! При перевыборе настройки из справочника  все изменения потеряются. Таким образом после отработки настроек рекомендуется внести соответствующие изменения в справочник НАСТРОЙКИ ОПУ.

11. Справочник НАСТРОЙКИ ОПУ имеет две закладки ОТЧЕТ и НАСТРОЙКИ, соответствующие закладкам отчета. Закладки НАСТРОЙКИ отчета и справочника полностью соответствуют друг другу. На закладке ОТЧЕТ справочника добавлено наименование строки на кыргызском языке.

12. Порядок заполнения справочника:

·        указывается обозначение настройки

·        заполняется список строк отчета на закладке ОТЧЕТ

·        для каждой первичной строки отчета на закладке НАСТРОЙКИ указывается список счетов с формулами и знаками

·        для облегчения заполнения можно заполнить закладки справочника шаблоном по кнопке БАЗОВЫЕ НАСТРОЙКИ и затем отредактировать закладки

13. Если на закладке ДОПОЛНИТЕЛЬНО установлен флажок «Общий отчет», то по кнопке ПЕЧАТЬ будет выводиться произвольный отчет с наименованием, также заданным на закладке ДОПОЛНИТЕЛЬНО. При отсутствии флажка выводится стандартная форма отчета о прибылях и убытках.

14. Порядок формирования документа

·        открыть документ «Отчет ОПУ»

·        задать отчетный период

·        при необходимости уточнить предыдущий период

·        выбрать настройку в поле НАСТРОЙКА

·        по кнопке РАССЧИТАТЬ сформировать отчет

·        по кнопке ПЕЧАТЬ вывести отчет на печать

 

15. Проверка отчета о прибылях и убытках

 

1.       На закладке «Настройки»:

1.1.                Проверить, есть-ли счета доходов и расходов в оборотно-сальдовой ведомости, не учтенные в настройках – кнопка «Проверить настройки»

1.2.                Каждый счет дожен входить только в одну строку отчета - кнопка «Проверить настройки»

1.3.                В следущих строка не должны указываться счета:

 

·        009

·        030

·        039

·        060

·        069

·        150

·        160

·        180

·        200

1.4.                 В остальных строках обязательно должен быть указан счет

1.5.                 В колонке «Формула»:

·         для счетов доходов должно быть  - «ОборотыКт»

·         для счетов расходов должно быть          - «ОборотыДт»

·         для контрсчета 6120                       - «ОборотыДт», знак минус

·         для контрсчета 7220                       - «ОборотыКт», знак минус

16. Проверка настроек баланса

 

1.       На закладке «Настройки»:

1.1.  Проверить, есть-ли счета доходов и расходов в оборотно-сальдовой ведомости, не учтенные в настройках – кнопка «Проверить настройки»

1.2. Каждый счет дожен входить только в одну строку отчета - кнопка «Проверить настройки» (кроме 5300)

1.3. В следущих строка не должны указываться счета:

 

098, 099, 130, 130a, 140, 140a, 150, 150a, 160, 160a, 170, 170a, 180, 180a, 190, 100, 192

210, 210a, 212, 220, 220a, 230, 230a, 240, 250, 270, 270a, 280, 280a, 290a, 200, 299, 299а, 299б

330, 330a, 340, 340a, 345, 350, 350a, 353, 300, 399

410, 410a, 420, 400, 499, 499а,

510, 510a, 520, 520a, 500, 599

 

1.4.  В остальных строках обязательно должен быть указан счет

1.5.  Для счетов на 1 и 2 в колонке «Дебет» повторяется колонка «Счет»,  формула «Сальдо Дт»

1.6.  Для счетов на 3, 4 и 5 в колонке «Кредит» повторяется колонка «Счет», формула «СальдоКт».
