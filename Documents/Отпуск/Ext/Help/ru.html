Документ «Отпуск»

 

Документ предназначен для оформления отпуска сотруднику. Создается на каждого сотрудника отдельно.

Указать вид отпуска, дату начала отпуска. На закладке «Дополнительный отпуск» в первой строке указываем

основной вид отпуска, в следующих строчках  дополнительные виды отпусков и количество дней.  

       Закладки «Средний заработок» заполняется автоматически при нажатии кнопки «Заполнить», можно

 добавлять, редактировать строку. На закладке «Начисления»  по кнопке «Рассчитать»  автоматически

 рассчитываются начисленные суммы отпускных за указанный период, с разбивкой по видам и месяцам отпуска.

Неполный месяц

ВР

1.Не изменять – для расчета отпускных выводить фактически отработанные часы  и начисленную сумма
2. Дополнять  - для расчета отпускных выводить норм часов и размер по основному виду начисления
3. Отбрасывать  - в расчете отпускных не учитывать если есть неявки, больничные и отработал меньше нормы часов.

Указать виды премии, которые используются для расчета отпускных.

 

·         В - видимость

·         РВ - ручной ввод

·         УМ - по умолчанию

Расчет отпускных производится по следующей формуле:
            ((начисления за 3 предыдущих месяца/отработанные часы за 3 месяца)*Коэфф. расчета для начисления+ 1/12премия) * кол-о дней отпуска.
       - Начисления за 3 предыдущих месяца – сумма начислений  указывается на закладке «Средний заработок»
       - отработанные часы за 3 месяца – указать количество месяцев для расчета среднего заработка для каждого вида отпуска в

Планов видов расчета  «Вид Начисления» в поле «Период расчета для среднего заработка»

    -  Коэфф. расчета для начисления – указать в справочнике «Вид отпуска» для каждого вида отпуска

    -  1/12премия -  Заполняется сумма по каждому виду премии необходимая для расчета 1/12 премии.

   - кол-о дней отпуска -  дни отпуска указываются для каждого вида отпуска и количества месяцев в периоде отпуска.
