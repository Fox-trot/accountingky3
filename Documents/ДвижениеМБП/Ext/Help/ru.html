Документ Движения МБП предназначен для отражения в учёте операций движения МБП.

Документ содержит следующие виды операций:

·                    Ввод МБП в эксплуатацию;

·                    Возврат на склад;

·                    Перемещение МБП;

·                    Перемещение МБП в эксплуатации;

·                    Списание из эксплуатации;

·                    Списание со склада;

·                    Реставрация.

 

Вид операции Ввод МБП в эксплуатацию предназначен для отражения в учёте ввода МБП в эксплуатацию. Указывается Склад, МОЛ, Счет затрат и статья затрат. МОЛ подбирается со справочника «Физ. Лица», склад со справочника «Склады/места хранения», счет с Плана счетов, а статья затрат со справочника «Статьи затрат». Табличная часть заполняется по командам «добавить» или «подбор». Если используется команда «добавить», то при вводе номенклатуры, количество заполняется всеми остатками, которые есть на складе, далее пользователь вводит нужное количество. Если пользователь воспользуется командой «подбор», то в подборе будет отражены все остатки которые, есть на складе, затем подобранная номенклатура переносится в ТЧ документа. Если учет МБП ведется количественно, то в РС Учетная политика организации необходимо установить флажок «количественный учет МБП». При количественном учёта инвентарные номера не присваиваются. Если же учет МБП осуществляется по инвентарным номерам, то РС Учетная политика организации снять флажок «количественный учет МБП» и документе после подбора номенклатуры нажать на команду «дать инвентарные номера», по этой команде программа присвоит свободные инвентарные номера.

 

Вид операции Возврат на склад предназначен для тех случаев, когда ранее МБП был введён в эксплуатацию, а затем обратно вернули на склад (возможно уже не нужен, или же в МБП нет необходимости). После возврата МБП на склад, МБП можем заново ввести в эксплуатацию.

Заполняется склад, МОЛ, далее по команде «подбор» выбирается МБП, которую нужно вернуть на склад.

 

Вид операции Перемещение МБП предназначен для отражения в учёте перемещение МБП с одного склада на другой склад. Заполняется склад и склад получатель. Табличная часть заполняется по командам «добавить», «подбор», «заполнить». По команде «добавить» добавляется строка, и указывается количество. По команде «подбор» открывается форма подбора номенклатуры со склада. По команде «заполнить» ТЧ заполняется всеми остатками склада отправителя.  

Вид операции Перемещение МБП в эксплуатации предназначен для отражения в учёте перемещение МБП с одного подотчетника на другого (увольнение, отпуск и т.д.). Заполняется МОЛ и МОЛ получатель. Заполнение ТЧ аналогично заполнению ТЧ с видом операции Перемещение МБП.  

Вид операции Списание из эксплуатации предназначен для списания МБП с подотчёта МОЛ  (непригодность, износ и т.д.). Заполняется МОЛ, ТЧ заполняется по команде «добавить», «Подбор».  В ПФ комиссия выводится с закладки Комиссия, которая заполняется по команде «добавить», либо с помощью правой кнопки мыши, далее «Добавить». 

Вид операции Списание со склада предназначен для отражения  в учёте списания номенклатур со склада (кража, порча). Заполняется Склад, счет затрат и статья затрат. По команде подбор пользователь выбирает номенклатуру.

 

Вид операции Реставрация предназначен для тех случаев, когда списываем одну номенклатуру и получаем другую. Пример списываем простыни, получаем одеяла. Списать номенклатуру можем, которая на складе, так и те, которые в эксплуатации. Если реставрируется со склада, то в поле Реставрация нажимаете на «склад» (цвет будет зелёный), и указываете склад, по команде «подбор» выбираем номенклатуру. Далее в ТЧ Реставрация заполняем строку номенклатурой, которую получили от реставрации. Если реставрируем с эксплуатации, то тогда в поле реставрация нажимаем на «эксплуатация (цвет будет зелёный) и указываете МОЛ. Затем также заполняете ТЧ Реставрация.
