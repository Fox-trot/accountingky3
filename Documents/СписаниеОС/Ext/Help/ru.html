Документ «Cписание ОС»

Документ «Списание ОС» предназначен для отражения в учете выбытия основного средства:

1. в связи с износом

2. списание на ремонт

3. списание на капитализацию

4. списание с ускоренной амортизацией

5. списание для передачи

 

1) Если в уч.политике установлен флажок Расчет амортизации по месяцам, то при Списании ОС начисляется амортизация следующим образом:

1. Списание ОС до окончания срока полезной службы:

• Начисляется амортизация за месяц (первонач.ст-ть\СПС = норма аморт.) 

Дт 8ххх Кт 219х ххх;

• Доначисляется остаточная стоимость (первонач.ст-ть - накоп.аморт. = остат.ст-ть)

Дт 8ххх Кт 219х ххххх ;

• Списывается первоначальная стоимость (первонач.ст-ть) Дт 219х Кт 219хх ххххх.

2. Списание после окончания СПС:

• Списывается первоначальная стоимость Дт 219х Кт 219х ххххх

 

2) Если в уч.политике не установлен флажок, то начисление амортизации следующее:

1. Списание ОС до окончания срока полезной службы:

• Начисляется амортизация по дате списания ОС (норма аморт\кол-во дней в месяце*дни по дате списания) Дт 8ххх Кт 219х ххх;

• Доначисляется остаточная стоимость (накоп.аморт.+аморт.до даты списания)

Дт 8ххх Кт 219х ххх;

• Списание стоимости ОС (первонач.ст-ть) Дт 8ххх Кт 219х ххх.

2. Списание после окончания СПС:

• Списывается первоначальная стоимость Дт 219х Кт 219х хххх

 

 

Наименование

 

Содержание

В шапке

 

 

Номер

Авто, РВ

Автоматически присваивается номер документа, можно отредактировать вручную

От

Авто, РВ

Дата документа автоматически устанавливается дата создания документа, можно отредактировать вручную

Организация

УМ

По умолчанию устанавливается основная организация, указанная в справочнике организации. Если ведется учет по нескольким организациям, можно изменить организацию  из справочника «Организация»

Документ основание

 

Указать документ на основании, которого списывается основное средство

Событие

 

Указывается событие «Списание ОС»

МОЛ

 

Выбор из справочника физ.лица

Подразделение

 

Выбирается из списка подразделений по нажатию на кнопку выпадающего списка   . Также по нажатию в поле на  открывается окно редактирования данных по подразделению.

Закладка Основные Средства

 

 

Кнопка Подбор ОС

 

Из справочника выбираем списываемое ОС.

Для каждого объекта основных средств автоматически заполняются строки в ТЧ:

- первоначальная стоимость,

- ликвидационная стоимость

- остаточная стоимость (первонач.ст. – накопленный износ)

- накопленная амортизация

- амортизация/мес, норма амортизации,

- счет учета,

- статья затрат,

- счет расходов.

Кнопка Заполнить

 

Используется для быстрого заполнения табличной части однотипными объектами основных средств

Закладка Комиссия

 

 

Причина списания

 

при нажатии на кнопку   в появившемся окне «Причина списания ОС» нужно создать новый параметр.

 

Техническое состояние

РВ

Описать техническое состояние. Ввод вручную

ТЧ
