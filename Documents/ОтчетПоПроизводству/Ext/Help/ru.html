Отчет по производству

Документ предназначен для регистрации операций, выполненных по заказам.

Документ доступен при установленном флажке Учет производства по заказам (Настройки – Параметры учета - Производство).

В документе может отражаться закрытие различных операций по различным заказам.

Закладка Готовая продукция служит для отражения выпуска готовой продукции:

  • Контрагент- заказчик готовой продукции.
  • Заказ- документ, предназначенный для регистрации заказа на производство.
  • Номенклатура- наименование готовой продукции.
  • Спецификация
  • Вид работы
  • Количество- количество готовой продукции.
  • Склад- на который приходуется готовая продукция.
  • Бригада- подразделение, выполняющее заказ.

Флажок Полуфабрикаты устанавливается, если заказ на изготовление полуфабрикатов.

Закладка Материалы отражает списание материалов в производство:

  • Контрагент- заказчик готовой продукции.
  • Заказ- документ, предназначенный для регистрации заказа на производство.
  • Продукция- наименование готовой продукции.
  • Вид работы
  • Номенклатура- материалы для готовой продукции.
  • Количество- количество материалов, расходуемые на изготовление готовой продукции.
  • Счет учета- счет, на который приходуются материалы.
  • Склад- склад, на который относится готовая продукция.

Флажок Полуфабрикат устанавливается, если заказ на изготовление полуфабрикатов.

По кнопке Заполнить автоматически заполняется табличная часть товаров по видам работ из табличной части Готовая продукция.

Как найти: Производство → Документы → Отчет по производству
