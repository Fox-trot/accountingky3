Документ "Перемещение ОС"

Документ можно создать в разделе "ОС и НМА" - "Основные средства" - "Перемещение ОС".

Документ "Перемещение ОС" предназначен для внутреннего перемещения ОС между подразделениями или материально-ответственными лицами.

Реквизиты для заполнения:

- Организация;

- МОЛ (отправитель) – материально - ответственное лицо, за которым числится данное основное средство;

- МОЛ (получатель) – материально - ответственное лицо, которому передается данное основное средство. Выбирается из списка работников выбранной организации. При этом в форме выбора отображаются работники, которые на момент оформления документа приняты на работу в организацию;

- Подразделение (отправитель) – выбирается из справочника "Подразделения", на котором числится данное основное средство;

- Подразделение (получатель) – это подразделение, в которое перемещается основное средство.

При начале выбора подразделения открывается список подразделений только той организации, которая указана в документе.

При проведении документа данные записываются в регистр сведений "Местонахождение ОС".

Проводки по бухгалтерскому учету не формируются.

Для быстрого заполнения табличной части однотипными объектами основных средств, имеющими одинаковые наименования, нужно ввести в табличную часть хотя бы один такой объект. Затем следует воспользоваться кнопкой "Заполнить" в командной панели табличной части. При выборе в выпадающем меню пункта "По наименованию" табличная часть будет заполнена объектами основных средств, имеющими такое же наименование, как у введенного первоначально.

Документ формирует печатную форму "Накладная".
