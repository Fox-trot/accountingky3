Документ "Инвентаризация ОС"

Документ можно создать в разделе "ОС и НМА" – "Основные средства" – "Инвентаризация ОС".

Цель инвентаризации основных средств заключается в выявлении фактического наличия основных средств и проверке соответствия данных бухгалтерского учета по количеству и стоимости основных средств фактическому наличию и последующем отражении в учете принятия обнаруженных объектов и (или) выбытия не обнаруженных.

Данные учета заполняются по данным информационной базы при помощи кнопки "Заполнить – по остаткам" в командной панели табличной части "Основные средства".

Выявленные в ходе инвентаризации объекты основных средств можно добавить в табличную часть при помощи кнопки "Добавить".

Для основных средств, по которым не заполнены данные учета, можно ввести фактическую стоимость.

Фактическая стоимость основных средств, которые числятся в учете, считается равной учетной. Редактировать ее нельзя.

Для проведения инвентаризации в организации создается постоянно действующая инвентаризационная комиссия, состав которой утверждается распоряжением (приказом) руководителя.

Для того, чтобы указать состав инвентаризационной комиссии, в документе необходимо перейти на закладку "Дополнительно" и заполнить необходимые поля. Для заполнения табличной части "Состав комиссии", данные выбираются из справочника "Физические лица".

Документ "Инвентаризация ОС" в бухгалтерском учете не отражается, однако, в случае расхождений, выявленных в результате инвентаризации (излишки или недостачи) на основании документа "Инвентаризация ОС" можно сформировать документы:

- "Принятие к учету основных средств" - по которым в результате инвентаризации выявлен излишек. Для этого необходимо использовать документ "Принятие к учету ОС" с видом операции "По результатам инвентаризации"

- "Списание основных средств" - по которым в результате инвентаризации выявлена недостача. Для этого следует воспользоваться документом "Списание ОС".

Результаты инвентаризации должны быть отражены в учете и отчетности того месяца, в котором была закончена инвентаризация.

Документ "Инвентаризация ОС" формирует печатные формы инвентаризационной описи (форма ИНВ-1) и сличительной ведомости (форма ИНВ-18) по кнопке "Печать".
