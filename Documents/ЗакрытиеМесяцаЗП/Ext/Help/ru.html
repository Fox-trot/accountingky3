Закрытие месяца по ЗП

Расчет налогов производится с помощью документа Закрытие месяца по зарплате.

Документ предназначен для расчета подоходного налога, соц.отчислений с работников и работодателей, плановых удержаний.

Для одной организации документ оформляется один раз в месяц. Этим же документом формируются все проводки по всем начислениям и всем удержаниям по организации.

В шапке документа необходимо выбрать период.

По кнопке Выполнить закрытие месяца формируются две закладки документа: Налоги по заработной плате, Удержания.
На закладке Налоги по заработной плате для каждого работника организации приведены итоговые начисления и налоги.
На закладке Удержания приведены все плановые удержания.
На закладке Дополнительно указывается комментарий к документу.

Как найти: Зарплата и кадры → Закрытие месяца → Закрытие месяца по зарплате
