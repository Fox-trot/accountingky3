Документ "Модернизация ОС"

Документ предназначен для оформления:

1. Модернизация ОС – увеличение первоначальной стоимости (в поле Стоимость модернизации по кнопке

Рассчитать подставляется дебетовый остаток счета 2123 на дату документа по данному ОС);

2. Изменение параметров (возможно при увеличении стоимости и когда стоимость не увеличивается).

 

В одном документе обрабатывается только одно основное средство.

 

Наименование

 

Содержание

Номер

 

Автоматически присваивается номер документа, можно отредактировать вручную

От

 

Дата документа автоматически устанавливается дата создания документа, можно отредактировать вручную

Организация

 

По умолчанию устанавливается основная организация, указанная в справочнике организации. Если ведется учет по нескольким организациям, можно изменить организацию  из справочника «Организация»

УМ

Счет учета 2123 - «Модернизация ОС»

Стоимость модернизации

 

По кнопке Рассчитать выводится стоимость модернизации по дебетовому остатку счета 2123 с субконто ОС указанного в шапке.

Событие

 

Если стоимость ОС увеличивается, то в поле Событие должно быть выбрано Событие с видом события Модернизация.

Если стоимость ОС не увеличивается, то в поле Событие должно быть выбрано Событие с видом события Изменение параметров.

Перед записью производится контроль правильности выбора События.

Сумма амортизации

 

Если в учетной политике установлен флажок расчет амортизации по месяцам, то начисляется амортизация за месяц по первоначальной стоимости на начало месяца. (В закрытии месяца амортизация по данному ОС не начисляется)

Если в учетной политике не установлен флажок. Амортизация за дни месяца до модернизации начисляется этим же документом по первоначальной стоимости на начало месяца.

Амортизация за дни месяца со дня модернизации начисляется в закрытии месяца по увеличенной первоначальной стоимости.
