Инвентаризация незавершенного производства

Документ предназначен для отражения остатков незавершенного производства.

Этот документ необходимо ввести по каждому производственному подразделению, в котором есть остатки НЗП на конец месяца.

В конце каждого месяца проводится инвентаризация незавершенного производства (НЗП), в случае, если производство не завершено полностью в текущем месяце, то вводятся суммы затрат, которые должны быть перенесены на следующий период.
Расходы, отнесенные к НЗП, не учувствуют в расчете себестоимости продукции (полуфабрикатов), выпущенной в текущем месяце и остаются в остатке по счету затрат (например: 1630).

Документ не формирует движений по регистру бухгалтерии.

Общие реквизиты документа (в шапке):

  • Дата и номер документа;
  • Организация;
  • Подразделение- подразделение, по которому вводятся остатки;

Закладка Остатки НЗП

  • Счет затрат- указывается счет затрат (Например: 1630 Основное производство).
  • Номенклатурная группа- номенклатурная группа, по которой вводятся остатки.
  • Сумма- сумма остатка по счету затрат, который переносится на следующий период. Вводится пользователем вручную.

На закладке Дополнительно указывается комментарий к документу.

Как найти: Производство → Документы → Инвентаризация незавершенного производства
