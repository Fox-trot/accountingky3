Отчет по НДС

Для формирования отчета необходимо указать период в поле Месяц.

Сформировать отчет можно по кнопке Заполнить. Отчет формируется после проведения документа Закрытие месяца за указанный период.

В отчете заполняются:

  • Основная форма
  • Приложение 1
  • Приложение 2
  • Приложение 3
  • Приложение 4,5 (если плательщик на основе налогового контракта)

В Приложении 1 Реестр поставок формируются данные по документам:

  • Реализация товаров и услуг
  • Возврат товаров от покупателя
  • Счета-фактуры (выписанные)

В приложении 2 Реестр приобретенных материальных ресурсов формируются данные по документам:

  • Авансовый отчет (закладка Патенты)
  • Поступление товаров и услуг
  • Возврат товаров поставщику
  • Счет фактура полученный

В Приложении 3 Реестр ввезенных материальных ресурсов в КР формируются данные по документам:

  • Поступление товаров и услуг (вид операции Покупка, у контрагента страна резидентства "Страны ЕАЭС")
  • ГТД по импорту

Приложение 4 Реестр поставок на основе налогового контракта заполняется данными документов соответственно Приложению 1.

Приложение 5 Реестр приобретенных на основе налогового контракта заполняется данными документов соответственно Приложению 2.

Приложение 4,5 заполняются вместо Приложения 1,2, если в Учетной политике организации установлен флажок Контракт.

По кнопке Выгрузка в excel отчет выгружается в формате excel.

На закладке Дополнительно указывается комментарий к документу.

Печатные формы

Для документа Отчет по НДС предусмотрены следующие печатные формы:

  • Отчет по НДС
  • Реестр поставок
  • Реестр приобретенных материальных ресурсов
  • Реестр ввезенных материальных ресурсов в КР
  • Реестр поставок на основе налогового контракт
  • Реестр приобретенных материальных ресурсов на основе налогового контракта
