Отчет по НСП

При формировании отчета указать период в поле Месяц.

Выбрать вид субъекта:

  • Малый- формируется за квартал без приложений к отчету;
  • Средний- формируется основная форма и приложения к отчету;
  • Крупный- формируется отчет за месяц и основная форма.

Сформировать отчет по кнопке Заполнить. В отчете формируются данные по документам:

  • Реализация товаров и услуг
  • Возврат товаров от покупателя

По кнопке Печать формируются печатные формы отчета:

  • Отчет по НСП
  • Приложение к отчету по НСП

В печатной форме отчета в строке В безналичной форме формируются данные по документам с установленным флажком Безналичный расчет.

На закладке Дополнительно указывается комментарий к документу.

Как найти: Отчеты → Налоговая отчетность → Отчет по НСП
