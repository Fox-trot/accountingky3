Документ «Поступление товаров и услуг». БП-3

 

Документ «Поступление товаров и услуг» предназначен для оформления поступления ТМЗ, услуг и ОС в результате закупок у поставщиков.

 

Наименование

Назначение

Замечания

Операции

 

 

Покупка

Основная операция покупки товаров, ОС и услуг у поставщика.

При открытии устанавливается по умолчанию (если не копирование)

Варианты операции определяются поставщиком: КР, ЕАЭС, Импорт (если страна КР, срана имеет признак ЕАЭС и страна не КР и не имеет признак ЕАЭС).

 

 

РБП

Операция поступления расходов будущих периодов (РБП). Сумма РБП поступает на счет 1880 (РБП). НДС оформляется как при операции поступления. Указывается РБП по справочнику РБП. Если предусмотрено списание РБП в текущем месяце, то такая проводка оформляется в ЗМ.

Операция РБП возможна только для услуг

 

 

Ввод на основании

Документ можно создать на основании документов: Доверенность, РКО, ППИ, АО

На основании документа можно создать: Доверенность, Реализация товаров и услуг, Движение МБП, РКО, ППИ, Возврат товаров поставщику, ГТД по импорту,   Допрасходы,        Счет-фактура полученный

Распределение акциза

Перечисление: По весу, По сумме, По количеству, По доп/единицам

 

Видимость:

Операция  Покупка и Контрагент ЕАЭС

 

Расчет при изменении (2 знака):

1. Распределение акциза (обеспечить стандартную компенсацию округления):

- По сумме = Всего  по строке * Сумма акциза / Сумма Всего по ТЧ

- По кол-ву= Кол-во ед. по строке * Сумма акциза / Общее кол-во

- По весу     = Вес по строке * Сумма акциза / Общий Вес

- По допЕИ = КолДЕ * Сумма акциза / Сумма КолДЕ

СчетУчета

Заполняется: При изменении номенклатуры и операции из РС СУН

 

Порядок ввода

1. Вводим Всего и количество

2. Рассчитываются все остальные суммы

3. Меняем цену – ничего не меняется

4. При изменении суммы компенсируем за счет НДС

5. При изменении НДС компенсируем за счет НСП

6. Колонка НСП – не доступна

 

 

Расчет строки Товары

Если количество 0, то подставляется 1

 

1. КР:

1.1.  При изменении Всего и Количество:

 

Сумма = Окр(Всего * 100 / (100 + Ставка НДС + Ставка НСП), 2)

НДС     = Окр(Всего * Ставка НДС /

                          (100 + Ставка НДС + Ставка НСП)  , 2)

НСП    = Всего – Сумма – НДС

 

Цена = Сумма/Количество, 2

 

1.2. При изменении цены: пересчет не производится

 

1.3. При изменении суммы: НДС = Всего – Сумма – НСП

 

1.4. При изменении НДС: НСП = Всего – Сумма - НДС

Колонка НСП – не доступна

 

 

2. ЕАЭС:

2.1. При изменении Всего и Количество:

Сумма Всего (РВ в валюте)

Сумма (не включает налоги) = Всего * Курс + Сумма акциза

НДС      = Окр(Сумма * Ставка НДС, )

НСП     = 0, колонка невидима

Цена =Всего /Количество, 2

Проводки:

Дт 1ххх  Кт 31хх (поступление тов-в, услуг, ОС) Валюта Кт (не реглам.)

Дт 1540 Кт 31хх (НДС)  Валюта реглам. (сом)

Дт 1ххх  Кт 3460 Акциз к оплате. Валюта реглам. (сом)

 

Если Освобожденный импорт, то ставка НДС = 0

 

2.2. При изменении цены: пересчет не производится

2.3.  Колонки Сумма и НДС недоступны

 

 

3. Импорт и НДС не подтвержден:

3.1. При изменении Всего и Количество:

Сумма = Всего, колонка невидима

НДС      = 0, колонка невидима

НСП      = 0, колонка невидима

Цена = Сумма/Количество, 2

 

 

3.2. При изменении цены: пересчет не производится

3.3. Заголовок колонки Всего = «Сумма»
