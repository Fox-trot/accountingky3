Поступление товаров и услуг

Документ Поступление товаров и услуг служит для отражения операций по поступлению товаров и услуг, основных средств в результате закупок от поставщиков.

При помощи документа Поступление товаров и услуг можно выполнить несколько видов операций. Устанавливается вид операции с помощью поля выбора Вид операции, расположенной в форме документа.

Для всех видов операции требуется заполнить поля:

  • Контрагент- поставщик;
  • Договор (должен иметь вид договора С поставщиком). Указывается договор на основании которого поступают товары, услуги или основные средства;
  • Склад- выбирается склад, на который будут приходоваться товары.

По умолчанию заполняются:

  • Операция – Покупка;
  • Склад – основной склад организации;

Операция Покупка выбирается в случае регистрации приобретения товаров, основных средств и услуг от поставщиков.

Операция Расходы будущих периодов (РБП)- сумма РБП поступает на счет 1880 (РБП). НДС оформляется как при операции поступления. Указывается РБП по справочнику РБП. Если предусмотрено списание МБП в текущем месяце, то такая проводка оформляется в документе Закрытие месяца. Операция РБП возможна только для услуг.

Признак Безналичный расчет устанавливается, если оформляется оплата через банк. В этом случае НСП не начисляется.

Так как документ является универсальным, то есть с помощью этого документа можно оприходовать и основные средства, и товары, и услуги, для этих целей предусмотрено несколько табличных частей.

На закладке Товары оформляются сведения о поступивших в организацию товарах.

В случае поступления основных средств, необходимо выбрать закладку Основные средства. По каждой поступившей позиции указывается наименование основного средства, единица измерения, количество, общая сумма, цена за единицу, сумма НДС, сумма НСП, признак зачета НДС и счета учета.

Если организации были оказаны услуги сторонними организациями, которые должны быть отнесены на затраты, то информация по этим затратам указывается на закладке Услуги. Для данного вида операции в поле Номенклатура табличной части документа может быть указан элемент справочника Номенклатура, для которого установлен флажок Услуга, и текстовое название приобретаемой услуги.

Импорт из стран ЕАЭС

Для того, чтобы оформить поступление импорта из стран таможенного союза для контрагента на закладке Реквизиты в поле Страна резидентства выбрать одну из стран:

  • Россия
  • Казахстан
  • Белоруссия
  • Армения

Виды зачета НДС

Для позиций поступления ТМЗ и услуг в табличных частях указывается признак Зачет НДС (возможные значения: Зачет, Себестоимость и Распределение). Для незаполненных позиций и для табличной части ОС признак принимает значение Зачет.
Если флажок Указывать признак зачета НДС в учетной политике не установлен, то признак Зачет НДС скрыт и принимает значение по умолчанию Зачет.

В зависимости от признака формируются следующие проводки по НДС:

  • Зачет - проводка типа Дт 1540 - Кт 3110, при закрытии месяца НДС не участвует в распределении, то есть НДС безусловно принят к зачету;
  • Себестоимость - проводка типа Дт1610 - Кт 3110 (НДС относятся на себестоимость ТМЗ), Дт 8XXX - Кт 7XXX (Услуги) в закрытии месяца НДС - не участвует в распределении - не принимается к зачету, то есть НДС включается в себестоимость;
  • Распределение - проводка типа Дт 1540 - Кт 3110, при закрытии месяца - сумма НДС с этим признаком распределяется;

Далее, при закрытии месяца рассчитывается сумма НДС, не разрешенного к зачету.

Налог с продаж (НСП)

При поступлении товаров и основных средств сумма налога с продаж включается в себестоимость, отдельная проводка не формируется (согласно 2-му стандарту МСФО все невозмещаемые налоги включаются в себестоимость).

На закладке Счет-фактура указываются реквизиты полученного счета-фактуры.

Признак Не включать в реестр счет-фактуры указывается, если данные не нужно отражать в отчетах НДС на закладке Реестр приобретенных материальных ресурсов.

На закладке Показатели импорт указываются показатели для отчетов по импорту ЕАЭС (например, Отчет по косвенным налогам).

На закладке Дополнительно указываются:

  • счета расчетов с контрагентами
  • значения ставок НДС и НСП
  • комментарий к документу

Использовать доп. единицы- при установленном признаке на закладке Товары добавляется колонка Количество ДЕ.

Ввод на основании

На основании документа Поступление товаров и услуг можно ввести следующие объекты:

  • Возврат товаров и услуг поставщику
  • ГТД по импорту
  • Дополнительные расходы
  • Платежное поручение (исходящее)
  • Принятие к учету ОС
  • Расходный кассовый ордер
  • Счет-фактура полученный
