1. Приход бланков СФ

Бланки СФ
  1. Указывается операция «Поступление»
  2. Номера бланков можно указать с первого по последний
  3. В регистр сведений «Серии бланков СФ»  записываются серия, формат, признак  ручной СФ
  4. В регистр накопления «Бланки СФ» для каждого бланка формируется приход

2. Утеря бланка СФ

Бланки СФ
  1. Указывается операция «Утеря бланков  СФ»
  2. Номера бланков можно указать с первого по последний
  3. В регистр накопления «Бланки СФ» для каждого бланка формируется расход

3. Испорчен бланк СФ

Бланки СФ
  1. Указывается операция «Порча бланков СФ»
  2. Номера бланков можно указать с первого по последний
  3. В регистр накопления «Бланки СФ» для каждого бланка формируется расход

4. Производственный брак бланков СФ

Бланки СФ
  1. Указывается операция «Производственный брак бланков СФ»
  2. Номера бланков можно указать с первого по последний

В регистр накопления «Бланки СФ» для каждого бланка формируется расход

5. Использование бланка СФ

Реализация ТМЗ и услуг;

СФ выписанный
  1. Выбирается бланк СФ – для одной счет-фактуры только один бланк
  2. В регистр накопления «Бланки СФ» для каждого бланка формируется расход
