Прием на работу

Документ предназначен для оформления приема на работу. Создается на каждого сотрудника отдельно.

На закладке Приказ заполнить сведения для приема на работу сотрудника: выбрать сотрудника, которого принимаем на работу, указать Подразделение, Должность, График работы, Статус.
В поле Дата приема указать дату приема сотрудника на работу.
В поле Дата по при необходимости заполняется дата окончания периода работы сотрудника.
В поле Испытательный срок автоматически устанавливается +3 месяца от даты приема. Возможно, отредактировать дату вручную. Если принимается на работу без испытательного срока дату не устанавливать.
В поле Размер указать размер оклада принимаемого сотрудника.
В поле Вид начисления указать основной вид начисления зарплаты.
В поле Доп.вычеты указать количество иждивенцев.
Предыдущий стаж- указать общий стаж по трудовой деятельности (лет, месяцев, дней).

Выбрать из справочника Подразделение, в которое принимается сотрудник.

Если у сотрудника несколько видов начислений, то добавить их на закладке Прочие начисления.

На закладке Дополнительно указывается комментарий к документу.

Печатные формы

Для документа Прием на работу предусмотрены следующие печатные формы:

  • Приказ о приеме на работу
  • Трудовой договор
