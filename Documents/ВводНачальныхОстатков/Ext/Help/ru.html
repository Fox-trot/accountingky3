Ввод начальных остатков

Документ используется для ввода начальных остатков.
Организации следует определить дату начала ввода остатков.
Раздел учета – отражается список по типам операции:

• Запасы
• Основные средства
• Расчеты с персоналом
• Денежные средства
• МБП
• Налоговый учет
• Прочее
• Авансы и доотгрузка

При вводе в шапке документа нужно указать:

• Дата – рекомендуется начинать ведение учета с начала года, квартала или месяца. Датой ввода начальных остатков должна быть дата последнего календарного дня.
• Организация
• Счет учета, на котором имеются остатки

В регистр бухгалтерии записываются проводки в корреспонденции со счетом 0000.

Раздел учета Запасы.

Закладка Товары - остатки по товарам (материалам) следует вводить отдельными документами для каждого склада и счета. При вводе количества и цены сумма считается автоматически, при вводе суммы - цена считается автоматически.

Закладка Товары - остатки по товарам (материалам) следует вводить отдельными документами для каждого склада и счета. При вводе количества и цены сумма считается автоматически, при вводе суммы - цена считается автоматически.

Для ввода остатков по заработной плате необходимо выбрать раздел учета Расчеты с персоналом.
Если остатки заносятся по каждому подразделению отдельным документом, то указывается подразделение.
Если остатки вводятся по организации в целом, то поле Подразделение необходимо оставить пустым.

При проведении документа в регистре бухгалтерии формируются проводки Дт 0000 - Кт 3520.

Раздел учета Основные средства.

Документ предназначен для ввода всех сведений об основных средствах необходимых для дальнейшего ведения учета. В документ вносятся начальные данные для ведения бухгалтерского учета по основным средствам.

Закладка Основные средства – выбрать актив, о которому будет вводиться сальдо;
Инв.№ - инвентарный номер ОС;
Первоначальная стоимость – первоначальная стоимость ОС на момент принятия к учету;
Ликвидационная стоимость – указывается ликвидационная стоимость ОС;
Накопленная амортизация – сумма накопленного износа на дату ввода остатков;
Срок службы (мес.) – общий срок полезного использования ОС (срок указывается в месяцах);
База амортизации – сумма для расчета износа ОС;
Счет учета – счет учета, на котором будет учитываться ОС по бухгалтерскому учету, например, счет 2140 - Оборудование;
Счет амортизации – счет учета, на котором будет учитываться сумма накопленной амортизации по бухгалтерскому учету, например, счет 2194 - Амортизация - оборудование;
Способ амортизации – способ амортизации, установленный для основного средства по бухгалтерскому учету. Если основное средство является амортизируемым, то способ начисления амортизации должен быть установлен в обязательном порядке;
Отражение расходов – способ отражения расходов по амортизации основного средства по бухгалтерскому учету. Указывается в том случае, если основное средство является амортизируемым.
Дата принятия к учету – дата, с которой актив был принят к учету;
Вид документа – название документа, по которому актив был принят к учету;
№ документа – номер документа, по которому актив был принят к учету;
МОЛ – материально-ответственное лицо, за которым числится ОС;
Подразделение – подразделение, в котором эксплуатируется ОС;
Способ поступления – способ поступления в организацию, например, приобретение;
Группа НУ – амортизационная группа по налоговому учету;
Группа имущества – выбирается группа имущества, если ОС облагается налогом на имущество.
Расчеты с персоналом – предназначен для ввода начальных остатков по зарплате и депоненту, а также для ввода данных предыдущих периодов, необходимых для расчета подоходного налога нарастающим итогом, для расчета баз больничных и отпускных.

Раздел учета Расчеты с персоналом.

Закладка Расчеты с персоналом – указать сотрудника, по которому имеется остаток и сумму остатка.

Раздел учета Денежные средства.

Закладка Денежные средства - выбрать банковский счет (кассу), по которому будет вводиться сальдо;
Сумма валютная – валютная сумма остатка;
Валюта – если для счета установлен признак ведения валютного учета, необходимо выбрать валюту.

Раздел учета МБП.

Закладка МБП - остатки по малоценным и быстроизнашивающимся предметам следует вводить отдельными документами для каждого склада. При вводе количества и цены сумма считается автоматически, при вводе суммы - цена считается автоматически.
Эксплуатируемые МБП – при установленном флажке в табличной части документа появятся новые столбцы для заполнения:

• МОЛ
• Сумма износа - которую необходимо заполнить, если по МБП есть начисленный износ.

Раздел учета Налоговый учет.

Закладка Налоговая амортизация - предназначена для ввода балансовых категорий за год (стоимость на начало года и амортизации предыдущего года по группам ОС в соответствии с НК КР, а по 5-й и 6-й группе по объектам).
Закладка Налоговая декларация - вводятся суммы убытков за предыдущие пять лет для уменьшения налогооблагаемой прибыли при расчете налога на прибыль за текущий и следующие годы.
Суммы убытков отражаются в форме единой налоговой декларации (основная форма – стр 057)

Раздел учета Прочее.

Раздел предназначен для ввода начальных остатков по Контрагентам, Подотчетным лицам, Налогам, Капиталу организации и Прочим остаткам.
Счет учета – счет учета, на котором будет учитываться сальдо по бухгалтерскому учету;
Вид сальдо - указать в какой части проводки должен отразиться счет (сальдо Дт или сальдо Кт).
Субконто - указать аналитику счета по бухгалтерскому учету;
Валюта - указать валюту, по которой имеется остаток;
Курс – указывается курс валюты, по которой имеется остаток;
Валютная сумма – если для счета установлен признак ведения валютного учета, то при вводе остатков необходимо указать валютную сумму;
Количество – количество остатка;
Сумма – сумма остатка.

Раздел учета Авансы и доотгрузка.

Раздел заполняется по кнопке Заполнить. Табличная часть заполнится данными по контрагенту, договору контрагента, суммами отгрузки и аванса из предыдущего закрытия месяца.

Как найти: Настройки → Ввод начальных остатков
