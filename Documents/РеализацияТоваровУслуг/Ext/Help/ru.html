Реализация товаров и услуг

Документ Реализация ТМЗ и услуг предназначен для отражения операций по реализации товаров, услуг, основных средств.

В состав документа входят:

  • Склад- склад, с которого будут списаны продаваемые товары, основные средства;
  • Контрагент- покупатель товаров, услуг или основных средств;
  • Договор контрагента- договор, по которому осуществляется реализация;
  • Сумма документа- общая сумма документа по итогам всех табличных частей;
  • Вид скидки (наценки)- существует возможность указать скидку:
  • без скидки;
  • процент общий- % скидки указывается в шапке документа и используется для всех закладок;
  • процент по строке- в этом случае скидки указываются в табличной части и рассчитывается только для строки, где указан процент;
  • сумма по строке- проставляется сумма скидки на каждую строку отдельно;

Признак Безналичный расчет устанавливается, если предполагается оплата через банк. В этом случае НСП не начисляется.

Табличная часть Товары - служит для отражения операций по реализации ТМЗ.

Табличная часть Услуги- служит для отражения операций по оказанию сторонним организациям услуг. При добавлении в справочник номенклатуры услуг, нужно в карточке номенклатуры поставить флажок Услуга.

Табличная часть Основные средства- предназначена для оформления продажи основных средств, а также для оформления факта передачи ОС другой организации. Документ отражается по бухгалтерскому, налоговому учету.

На закладке Счет-фактура заполняются данные счета-фактуры. На закладке указываеются серия бланка СФ и номер бланка СФ.
Счет-фактура выписывается только в том случае, если были оформлению операции по приходу бланков счетов-фактур документом Поступление бланков счетов-фактур. Если ранее были оформлены операции по приходу бланков счетов-фактур, то при выборе серии программа откроет доступные серии. Если счет-фактура в момент реализации не выписывается, то счета-фактуры на несколько документов Реализации товаров и услуг можно выписать в документе Счет-фактура (выписанный).

На закладке Дополнительно указываются:

  • счета расчетов с контрагентами
  • значения ставок НДС и НСП (товаров и услуг)
  • комментарий к документу

Печатные формы

Для документа Реализация товаров и услуг предусмотрены следующие печатные формы:

  • Расходная накладная
  • Счет-фактура НДС (001)
  • Счет-фактура НДС (002)
  • Счет-фактура без бланка НДС
  • Акт об оказании услуг
  • ТТН
  • ТТН в валюте

Ввод на основании

На основании документа Реализация товаров и услуг можно ввести следующие объекты:

  • Возврат товаров и услуг от покупателя
  • Платёжное поручение (входящее)
  • Приходной кассовый ордер
  • Счет от поставщика
  • Счет на оплату покупателю
  • Счет-фактура (выписанный)
