Выработка ОС

Если для основного средства используется производственный способ начисления амортизации, для расчета амортизации необходимо указать фактический объем продукции (работ). Для отражения данного показателя предусмотрен документ Выработка ОС.

В документе необходимо заполнить реквизиты:
• Организация
• Инв. №
• Основное средство
• Параметр выработки
• Количество

Табличная часть заполняется двумя способами:
• С помощью кнопки Добавить
• С помощью кнопки Подбор (из справочника Основные средства)

На закладке Основные средства для каждого основного средства, по которому амортизация начисляется производственным способом, указывается параметр выработки и объем выработки (количество).

Данный документ не формирует никаких проводок.

Как найти: ОС и НМА → Основные средства → Выработка ОС
