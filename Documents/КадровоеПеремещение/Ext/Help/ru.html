Документ «Кадровые перемещения»

 

Документ предназначен для оформления постоянного и временного перемещения. Создается

на каждого сотрудника отдельно. При выборе Сотрудника на закладке «Начисления» и «Удержания» 

заполняются действующие виды начислений и удержаний с признаком «Не изменять». Можно вручную

установить признак «Прекратить».

На закладке «Приказ» необходимо указать основной вид начисления, на закладке «Начисления»,

«Удержания» можно добавить новые строки плановых видов начислений и удержаний.

Если установлен флажок «Временное», то после окончании срока временного перемещения автоматически  

восстанавливаются данные предшествующего приказа: подразделение, должность, размер, график работы,

основной вид начисления, плановые виды начисления и удержания. Временное перемещение оформляется в

 период временного отсутствия сотрудника. 
       Оформить следующий документ Кадровое перемещение возможно после даты возврата сотрудника на

предшествующий приказ.

 

 

Наименование

 

Содержание

Номер

В

Автоматически присваивается номер документа, можно отредактировать вручную

От

В

Дата документа автоматически устанавливается дата создания документа, можно отредактировать вручную

Организация

В

По умолчанию устанавливается основная организация указанная в «Учетной политике Предприятия». Если ведется учет по нескольким организациям, можно изменить организацию  из справочника «Организация»

Сотрудник

РВ

Выбрать сотрудника на которого оформляем кадровое перемещение. Варианты выбора сотрудника:

1. Написание ФИО/Таб. номер вручную и выбор из списка возможных вариантов

2. Выбор из справочника физических лиц - по стрелке

Дата С

РВ

Указать дату перемещения сотрудника

По

РВ

Если временное перемещение указать дату окончания перемещения

Флажок «Временное»

РВ

При установленном флажке, по окончанию даты временного перемещения автоматически возвращает прежнее место работы, начисления и удержания.

Ученик

РВ

Если перемещаем ученика.
