Документ "ГТД по импорту"

Документ можно создать в разделе "Покупка и продажа" – "Покупка" – "ГТД по импорту".

Документ "ГТД по импорту" оформляется в том случае, если необходимо при поступлении импортных товаров отразить информацию о таможенных сборах и пошлинах, зафиксированных в грузовой таможенной декларации (при поступлении товаров из третьих стран).

В шапке документа необходимо указать:

Таможня - указывается элемент справочника "Контрагенты".

Указывается информация об объектах аналитического учета расчетов с таможней: выбирается договор, по которому учитываются расчеты по валютным и сомовым таможенным платежам.

Документ "ГТД по импорту" может быть оформлен в любой валюте.

Номер ГТД - указывается номер грузовой таможенной декларации, по которой необходимо ввести информацию о таможенных пошлинах и сборах. Номер ГТД заполняется вручную.

Грузовая таможенная декларация может иметь несколько разделов, в которые группируются товары с одинаковым порядком расчета таможенных платежей (ставка пошлины, порядок расчета НДС, наличие акциза, суммы доставки и сопровождения).

По умолчанию предполагается, что ГТД имеет только один раздел.

Признак "Разделы ГТД" устанавливается, если есть несколько разделов. При установке признака в форме документа будут присутствовать две табличные части.

В строках верхней таблицы отображается сводная информация о таможенных пошлинах и сборах по разделам ГТД. Каждая строка этой таблицы представляет собой соответствующий раздел ГТД.  При установке курсора на одну из строк - один из разделов ГТД, в дополнительных таблицах "Товары" и "ОС" отображается информация о товарах по данному разделу ГТД.

По каждому разделу ГТД может быть введена своя сумма таможенной пошлины, НДС, фактурной стоимости, суммы доставки, таможенного сбора, сопровождения и акциза.

Если ГДТ имеет только один раздел, то вся информация представляется в полях закладки "Разделы":

Наименование Описание Проводки
Фактурная стоимость (ФС)

В колонках табличных частей "Товары" и "ОС":

1.ФС = Курс * Сумма валютная

2. Фактурная стоимость: сумма ФС по обеим табличным частям (Товары и ОС)

Распределение сопровождения Выбор из списка (По сумме, по количеству, по весу)
 
Распределение акциза – сумма акциза может быть распределена на товары и основные средства одним из способов:

- По количеству – при способе распределения "По количеству", сумма акциза распределяется следующим способом: определяется коэффициент распределения как отношение количества номенклатурной позиции к общему количеству номенклатурных позиций. Полученный коэффициент распределения умножается на сумму акциза и получается сумма акциза по каждой номенклатурной позиции;

- По сумме – сумма акциза будет распределяться в зависимости от суммы номенклатурных позиций, указанных в табличной части документа. Коэффициент распределения определяется как отношение суммы номенклатурной позиции к общей сумме номенклатурных позиций. Полученный коэффициент распределения умножается на сумму акциза по каждой номенклатурной позиции;

- По доп. Единицам – сумма акциза будет распределяться по формуле:

Количество * Коэффициент (из документа поступления товаров и услуг)

По кнопке "Рассчитать", все суммы ГТД будут распределены по позициям табличных частей "Товары" и "ОС" пропорционально фактурной стоимости позиций. При этом суммы ГТД могут быть введены вручную.

Признак "НДС только на пошлину" устанавливается для основных средств, указанных на закладке "ОС", НДС начисляется на пошлину импортируемых ОС, доставку.

Список товаров и сведения о документах поступления и счетах учета таможенных платежей указываются на закладке "Товары и ОС".

Закладка "Допрасходы" заполняется по кнопке "Подбор допрасходов". Перед этим должны быть оформлены все документы "Дополнительные расходы", относящиеся к данному разделу ГТД.

Документы "Дополнительные расходы" оформляются на основании документа поставки – "Поступление товаров и услуг", в табличных частях "ОС" и "Товары"  этих документов должен быть указан документ "Поступление товаров и услуг" как документ партии. 

На закладке "Дополнительно" обязательно должен быть указан счет расчетов с контрагентом – таможней. По умолчанию счет выбирается из регистра сведений "Счета расчетов с контрагентами".

Склад – на который приходуется товар.

Распределение пошлины:

- По сумме – сумма пошлины будет распределяться в зависимости от суммы номенклатурных позиций, указанных в табличной части документа. Коэффициент распределения определяется как отношение суммы номенклатурной позиции к общей сумме номенклатурных позиций. Полученный коэффициент распределения умножается на сумму пошлины по каждой номенклатурной позиции;

- По количеству - при выбранном способе распределения затрат "По количеству", сумма пошлины распределяется следующим способом: определяется коэффициент распределения как отношение количества номенклатурной позиции к общему количеству номенклатурных позиций. Полученный коэффициент распределения умножается на сумму пошлины и получается сумма пошлины по каждой номенклатурной позиции.

Если установлен флажок "Ввод сумм ГТД вручную", то суммы сбора, пошлины и НДС доступны для редактирования.

По кнопке "Рассчитать" все суммы ГТД будут пересчитаны по ставкам, при этом введенные вручную суммы ГТД будут пересчитаны.

Документ "ГТД по импорту" является частью комплекта документов, относящихся к операции импорта товаров по определенному инвойсу. В этот комплект входят:

- документ "Поступление товаров и услуг"

- один или несколько документов "Дополнительные расходы"

- сам документ "ГТД по импорту"

Сам документ "ГТД по импорту" также может быть оформлен на основании документа поставки – "Поступление товаров и услуг" или заполнен по нему.

При установленном флажке "Распределение вручную" суммы нижней табличной части корректируются вручную.

Документ проводится по бухгалтерскому учету. Информация о счетах бухгалтерского учета, в дебет которых будут отнесены таможенные платежи, за исключением санкций таможни, заполняется в табличной части "Товары" (по соответствующему разделу ГТД), в колонке "Счет учета (БУ)".

Таможенный сбор, таможенная пошлина, сопровождение и акциз проводятся в дебет счета учета ТМЗ с кредита счета расчетов с таможней (1610 – 3110).

НДС проводится в дебет счета "НДС к возмещению" с кредита счета расчетов с таможней (1540 – 3110).
