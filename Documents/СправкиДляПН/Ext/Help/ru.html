Документ предназначен для отражения данных, влияющих на расчет и отчетность по налогам ЗП.

ГНС - по умолчанию налоговая отчетность сдается по ГНС Фирмы (справочник Организации).
Если для подразделения указана ГНС, то отчетность по данному подразделению в указанной ГНС
Если для сотрудника указана ГНС, то отчетность по нему - в указанной ГНС

 

Документ проводится по регистрам Статусы сотрудников и ГНС Подразделения

Данные могут меняться только с 1-го числа месяца (один раз в месяц)
