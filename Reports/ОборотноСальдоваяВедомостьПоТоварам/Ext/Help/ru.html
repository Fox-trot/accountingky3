Оборотно-сальдовая ведомость по товарам

С помощью отчета оборотно-сальдовая ведомость по товарам осуществляется контроль остатков по счету товара.

При формировании отчета указать период.

В шапке установить флажок Номенклатура и выбрать в поле товар, если отчет нужно сформировать по одному товару. Если поле пустое, то формируется по всем товарам.

В шапке установить флажок Склад и выбрать в поле склад, если отчет нужно сформировать по одному складу. Если поле пустое, то формируется по всем складам организации.

В шапке отчета при установленном флажке Счет учета устанавливается отбор по выбранному счету учета номенклатуры.

Заполняется отчет по кнопке Сформировать.

В отчете формируются следующие данные:
• Начальный остаток
• Обороты за период
• Конечный остаток

По кнопке Настройки можно установить следующие отборы:
• Период
• Счет учета
• Организация
• Номенклатура
• Склад

По кнопке Найти можно осуществить поиск, например, по слову.

По кнопке Печать можно распечатать отчет.

По кнопке Сохранить отчет можно сохранить на компьютере.

Как найти: Запасы → Отчеты → Отчеты по запасам → Оборотно-сальдовая ведомость по товарам
