Итоговая ведомость по группам ОС за месяц

Отчет предназначен для формирования ведомости по группма основных средств за определенный период времени.

Каждая строка отчета соответствует определенному основному средству. Все основные средства сгруппированы по группам и упорядочены по наименованию в порядке возрастания.

Для каждого основного средства показаны суммы первоначальной и ликвидационной стоимостей, суммы накопленного и начисленного износов.

Как найти: ОС и НМА → Отчеты → Отчеты по основным средствам → Итоговая ведомость по группам ОС за месяц
