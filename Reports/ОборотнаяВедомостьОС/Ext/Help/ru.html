Отчет "Ведомость поступления ОС"

Отчет предназначен для формирования ведомости по поступившим основным средствам по всем счетам (субсчетам) за определенный период времени.

Каждая строка отчета соответствует определенному основному средству. Все основные средства сгруппированы по счетам учета и упорядочены по наименованию в порядке возрастания.

Для каждого основного средства показана сумма поступления в регламентированной валюте.

В отчете можно установить следующие настройки:


Группировка
Отбор
Сортировка
Дополнительные данные
Оформление
