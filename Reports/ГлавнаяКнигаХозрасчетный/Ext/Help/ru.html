Отчет "Главная книга"

Отчет позволяет вывести обороты счетов в корреспонденции со счетами и сальдо на начало и конец периода с заданной периодичностью.

Главная книга строится по всем счетам бухгалтерского учета для выбранной организации.

Реквизиты

·         Период - необходимо указать период, за который формируется отчет.

·         Организация - можно указать как одну, так и несколько организаций; по данным бухгалтерского учета указанных организаций строится отчет.

Действия

·         Кнопка «Сформировать» - по кнопке выполняется построение отчета в соответствии с текущей настройкой.

·         Кнопка «Показать настройки» - предназначена для редактирования перечня текущих настроек отчета; в открывшейся форме выполняется формирование условий отбора для построения отчета; список может содержать дополнительные параметры.

·         Кнопки «Сохранить настройки», «Выбрать настройки» - позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.

Закладка Общие

1.      Выводить данные

1.1.   По группам счетов – для детализации отчета по группам счетов.

1.2.   По субсчетам – если флаг выставлен, будет произведен расчет развернутого сальдо по субсчетам.

1.3.   По группам кор.счетов – для детализации отчета по группам кор.счетов.

1.4.   По субсчетам кор.счетов - для детализации данных по субсчетам кор.счетов.

2.      Периодичность отчета

2.1.   Период - задает периодичность, с которой будут выводиться промежуточные итоги.

2.2.   Все периоды - если флажок установлен, будут выводиться и те периоды, за которые не было оборота.

2.3.   Разбивать по листам - при печати отчета данные каждого счета выводились в виде отдельного листа главной книги.

2.4.   Выводить заголовок - позволяет получить более компактный вид отчета, скрывая заголовок ведомости. Кнопки сохранения и восстановления значений позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.

2.5.   Выводить подписи – при включенном признаке в печатной форме выводятся подписи ответственных лиц организации.

3.      Оформление

3.1.   Разбивать по листам - при печати отчета данные каждого счета выводились в виде отдельного листа главной книги.

3.2.   Выводить заголовок - позволяет получить более компактный вид отчета, скрывая заголовок ведомости. Кнопки сохранения и восстановления значений позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.

3.3.   Выводить подписи – при включенном признаке в печатной форме выводятся подписи ответственных лиц организации.

Закладка Развернутое сальдо

Задаются правила расчета развернутого сальдо для остатков на начало и конец периодов, для чего в табличной части указываются:

·         Счет - бухгалтерский счет, для которого необходимо выводить развернутое сальдо.

·         Субсчета - если флажок установлен, будет произведен расчет развернутого сальдо по субсчетам.

·         По субконто - в данном поле задается список видов субконто, по которым необходимо произвести расчет развернутого сальдо.

Результат отчета может быть сохранен, а также распечатан.
