Регистр сведений «Учетная политика по персоналу"

В рамках конфигурации в учетной политике по персоналу предусмотрены такие опции как:

Закладка «Общие»

·         Продолжительность рабочего дня (для расчета отработанного времени в часах количеству отработанных дней) для формирования графиков работы;

·         Учет отработанного времени по табелю. Если флажок установлен, то в обязательном порядке заполняется документ «Табель», в котором указывается отработанное время по основному виду расчета и на основании которого заполняются документы «Начисление зарплаты». В этом случае не работает механизм вытеснения (то есть отпуска и больничные не вытесняют оклад и т.д.). Если флажок сброшен, то документ табель не используется. По желанию его можно заполнить по факту заполнения документов «Начисление зарплаты»;

·         Налоги по подразделениям. При установленном флажке становится видимой закладка «Пределы налогов», на которой для каждого подразделения указываются Минимальный расчетный доход, ГНС, Среднемесячная зарплата по СФ и отделение Соц.фонда.

При установленном флажке «Налоги по подразделениям», но не заполненной вкладке «Пределы налогов» по менее чем 2м подразделениям, флажок будет сброшен;

·         Валюта – указывается валюта, в которой могут быть установлены размеры некоторых видов расчета, например, оклады. Если в плане видов расчета «Виды расчетов ЗП» для данного вида расчета установлен флажок «Валюта», то размер в приказе воспринимается, как указанный в валюте. При начислении зарплаты этот размер в валюте переводится в сомы по курсу из регистра сведений «Курсы валют ЗП» (Меню Зарплата \ Справочные данные по зарплате \ Курсы валют ЗП), то есть все расчеты, отчеты и печатные формы показываются в сомах;

·         Начало расчетного месяца - день месяца, являющийся первым днем отчетного периода для расчета ЗП. Указывается при несовпадении с началом календарного месяца. Например, расчетный месяц ЗП может быть с 25-го по 24-е число.

·         Начисления ГНПФ (мужчины), начиная с дат рождения. Указывается дата возрастного ограничения, для отчета по отчислениям в Соц.фонд.

·         Начисления ГНПФ (женщины), начиная с дат рождения. Указывается дата возрастного ограничения, для отчета по отчислениям в Соц.фонд.

Закладка «П/налог»

·         Минимальный расчетный облагаемый доход (МРД).  В соответствии со ст. 163 НК КР «Размер минимального расчетного дохода в месяц устанавливается уполномоченным налоговым органом ежегодно   на основании   данных статистического органа   Кыргызской   Республики по   региональной среднемесячной заработной плате сотрудников по состоянию на 1 октября предыдущего   года   с учетом стандартных вычетов, предусмотренных пунктами 1 и 3 статьи 170 настоящего Кодекса…».

·         Доплата ПН за счет фирмы. Если флажок установлен, то сумма подоходного налога, доплачиваемая до значения, соответствующего минимальному расчетному доходу, относится на расходы организации. Если флажок не установлен, то доплата производится за счет сотрудника;

·         Признавать расходы по доплате ПН в НУ. При установленном флажке расходы по доплате подоходного налога признаются в налоговом учете.  Если не установлен флажок, то доплата не признается в Налоговом учете.

·         Счет расходов по доплачиваемому ПН. Если установлен флажок «Доплата ПН за счет фирмы», то доплачиваемая сумма подоходного налога, относится в дебет указанного счета;

·         Статья по доплате ПН.  Если установлен флажок «Доплата ПН за счет фирмы», то доплачиваемая сумма подоходного налога относится на указанную статью.

 

Закладка «Соцотчисления»

·         Счет расчетов пенсионного фонда - бухгалтерский счет учета расчетов с ПФ (3531). Назначается как кредитовый счет проводок по начислению отчислений в пенсионный фонд с фирмы;

·         Счет расчетов мед. страхования - бухгалтерский счет учета расчетов с ФОМС (3532). Назначается как кредитовый счет проводок по начислению в ФОМС;

·         Счет расчетов фонда оздоровления трудящихся - бухгалтерский счет учета расчетов с ФОТФ (3533). Назначается как кредитовый счет проводок по начислению отчислений в пенсионный фонд с фирмы;

·         Счет расчетов Государственного накопительного пенсионного фонда - бухгалтерский счет учета ГН ПФР (3534). Назначается в проводках по начислению отчислений в пенсионный фонд с сотрудника;

·         Среднемесячная заработная плата (СЗПСФ). Ставка среднемесячной заработной платы меняется ежегодно с 1 июля текущего года. После изменения ставки, необходимо оформить новую учетную политику, указав размер новой ставки СЗПСФ.

·         Процент от СЗП. Указывается размер процента от среднемесячной заработной платы.

·         Счет расходов по корректировке СФ. Суммы доплат по соц. отчислениям с фирмы относятся в дебет указанного счета;

·         Статья расходов по корректировке СФ. Указывается статья расходов, выбираемая из справочника «Статьи затрат»;

·         Количество сотрудников для отчета СФ – метод определения количества сотрудников для отчетов по соц. отчислениям (по приказам или по начислениям).

Закладка «Дополнительно»

·         % корпоративных отчислений (профсоюзного взноса) – указывается ставка в процентах (например, 1%). Указывается, если все сотрудники уплачивают одинаковый процент корпоративных отчислений. Дополнительно при оформлении приказов для сотрудников на закладке «Сотрудники» должен быть указан признак «Корп.отч»;

·         Корпоративные отчисления за вычетом ПН. Если флажок установлен, то при расчете корпоративных отчислений будет отнят размер подоходного налога сотрудника в расчетном периоде;

·         Корпоративные отчисления за вычетом ПФР. Если флажок установлен, то при расчете корпоративных отчислений будет отнят размер пенсионного фонда сотрудника в расчетном периоде;

·         Счет расчетов по корпоративным отчислениям. Указывается счет расчета по профсоюзным отчислениям.
