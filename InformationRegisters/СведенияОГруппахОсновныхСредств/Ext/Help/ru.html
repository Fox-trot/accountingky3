Сведения о группах основных средств

В регистре сведений содержатся сведения о параметрах налогового учета основных средств, таких как предельная норма амортизации, процент для вычета суммы расходов на ремонт.

Регистр содержит список амортизационных групп основных средств в налоговом учете.

Сведения должны быть указаны для каждой группы основных средств, по которым ведется налоговый учет. Для активов пятой группы Здания, сооружения, строения, по которым учет ведется по каждому объекту отдельно, и в случае, если параметры налогового учета отличаются от параметров, которые установлены для группы в целом, необходимо создать значения для каждого объекта пятой группы.

Как найти: ОС и НМА → Прочее → Сведения о группах основных средств
